# Consuming Weather api from open map weather api

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

What things you need to install the software and how to install them

```
- requests
- docopt
```

### Installing

How to get the project up and running on a local machine 

Steps


```
- Clone or download the project to your local machine
- Navigate to the folder that contains run.py file
    - $ cd weather
- The run your program using python
    -$ python run.py -i
```

## Running the tests

To runs tests
```
$ pytest test_weather.py -v

```
